タイル

g.Tileでタイルチップを並べて描画できます。
画面をクリック(タップ)するとスクロールを開始／停止します。

このサンプルで使用している素材は素材ページで公開・配布しています。

  • game.json
  • mainScene.js
{
	"width": 512,
	"height": 384,
	"fps": 30,
	"assets": {
		"mainScene": {
			"type": "script",
			"path": "script/mainScene.js",
			"global": true
		},
		"map": {
			"type": "image",
			"width": 512,
			"height": 512,
			"path": "image/map.png"
		}
	}
}
